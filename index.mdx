GoodBoyDoc is a lightweight Python library that extracts and normalizes docstrings from your source code, then renders them in the output format of your choice. Whether you prefer Google-style, NumPy-style, or Sphinx-style docstrings, GoodBoyDoc handles the parsing and gives you clean, formatted documentation with a simple three-function API.

## Get started in three steps

<Steps>
  <Step title="Install GoodBoyDoc">
    Install the package from PyPI using pip:

    ```bash
    pip install goodboydoc
    ```
  </Step>
  <Step title="Generate docs from source">
    Pass your Python source code to `generate()` and pick a docstring style:

    ```python
    from goodboydoc import generate, format_output

    source = open("mymodule.py").read()
    docs = generate(source, style="google")
    print(format_output(docs, fmt="markdown"))
    ```
  </Step>
  <Step title="Configure your project (optional)">
    Create a `.goodboydoc.json` in your project root to set defaults:

    ```json
    {
        "style": "google",
        "output_format": "markdown",
        "include_private": false,
        "max_line_length": 79
    }
    ```
  </Step>
</Steps>
